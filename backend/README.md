# Chatbot REST API

## Technologies used
  - Node
  - Typescript
  - Express
  - Typeorm
  - Socket.io
  - Ejs


## Install dependencies
  yarn install

## Run the app
  yarn dev

### Request

`POST /settings`
`POST /users`
`POST /messages`
`GET /messages/:user_id`
`GET '/pages/client'`
`GET '/pages/admin'`
